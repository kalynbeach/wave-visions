# wave-visions

Audio => Data => Visions

## Usage

Install dependencies:

```bash
npm install
# or
pnpm install
# or
yarn install
```

Run development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

## Notes

### Concepts

- "Visions" -> live audio visualizers generated from audio/video inputs
  - Three.js
    - 3D scenes

### Libraries

- [three.js](https://github.com/mrdoob/three.js/)
  - [Docs](https://threejs.org/docs/)
